# How to submit a new language (Pull Request)

Fork this repository.
In your fork, click Add file → Upload files and upload your language file (example: es.json and ONLY use language codes).
Add a short message like: Add Spanish (es) translation and Commit changes.
Go back to your fork and click Contribute → Open pull request.
Set the title to: ``Add <Language Name> <code> translation`` and briefly describe what you added.
Click Create pull request.
That’s it — your translation will be reviewed and merged.
